To create your own models please use the Dockerfile-dataset to download and run the project with the dataset included.

	
	sudo docker build -t ids -f Dockerfile-dataset .

If your building from source please download the dataset and place it in the dataset directory.

	
	sudo apt-get install awscli
	aws s3 cp --no-sign-request --region  s3://cse-cic-ids2018/Original Network Traffic and Log data/Thursday-15-02-2018/pcap.zip .
	unzip pcap.zip
	mv pcap /path_to_project/dataset/
